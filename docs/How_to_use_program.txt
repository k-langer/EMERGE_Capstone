-Instruction to run the program
1. make sure the mysql is running
2. make sure leap motion and robot are connected to your machine via USB
3. sh build.sh //build all need source file
4. sh central.sh //start all processor

-To stop the program
1. sh stop.sh

-Log files

in the log directory, there are: lm_data.log, updater.log, controller.log

lm_data.log // log generated by leapmotion data collector
updater.log // log generated by arm_updater
controller.log // log generated by arm_controller

tail -f *.log //this will stream the all log files into your command line
tail -f lm_data.log // this will stream just lm_data into your command line
vi lm_data.log //to examine the log file
